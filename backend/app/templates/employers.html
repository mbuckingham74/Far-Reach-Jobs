Get your job postings in front of candidates seeking opportunities in remote Alaska and rural communities — completely free.

Bulk Import Job Sources

Have multiple job sources to submit? Upload a CSV file with your organizations' career pages and we'll review them for inclusion.

Limitations

  • Maximum 100 sources per submission
  • Maximum file size: 512KB
  • Sources are reviewed before being added
  • Each site requires manual configuration for scraping

Required columns: Organization, Base URL. Optional: Careers URL

Your CSV should look like this:

Organization,Base URL,Careers URL
City of Bethel,https://www.cityofbethel.net,https://www.cityofbethel.net/jobs
NANA Regional Corporation,https://nana.com,https://nana.com/careers
Yukon-Kuskokwim Health Corp,https://www.ykhc.org,

Column names are flexible (e.g., "Source Name" works for Organization, "URL" works for Base URL).

We may contact you if we have questions about any of the sources
