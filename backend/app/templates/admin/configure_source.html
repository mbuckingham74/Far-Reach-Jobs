How to Find CSS Selectors

  1. Open the job listings page in your browser
  2. Right-click on a job listing and select "Inspect" or press F12
  3. Find the HTML element that wraps each job - this is your Job Container
  4. Within that container, find elements for title, link, location, etc.
  5. Use CSS selectors like .class-name, #id, or tag.class

Example Selectors:

  • .job-listing - selects elements with class "job-listing"
  • h2.title a - selects links inside h2 elements with class "title"
  • [data-job-id] - selects elements with a data-job-id attribute
  • tr.job-row td:nth-child(2) - selects the 2nd column of table rows
